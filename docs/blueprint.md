# **App Name**: LuxuryEstate Nexus

## Core Features:

- Dynamic Hero Section: A prominent hero area featuring an autoplay muted video background, a bold headline overlay, and a clear call-to-action button, conveying an immediate sense of luxury and engagement.
- Interactive Property Listings: A responsive grid displaying premium property cards with engaging hover effects like image zoom, subtle scaling, and shadow lifts, to highlight desirable details.
- Advanced Filtering System: An intuitive filter bar incorporating a price range slider, location dropdown, and property type toggles, allowing users to efficiently narrow down property options.
- Persistent Theme Toggle: A user-controlled dark/light mode toggle with preference persistence via `next-themes`, ensuring a personalized and consistent viewing experience.
- Informative Footer Navigation: A minimalist footer section providing essential navigation links and social media icons, maintaining a clean and professional appearance.
- Mock Property Data Module: Localized mock data via a `data/properties.ts` file containing realistic high-end property listings for development and demonstration purposes, without requiring a database.

## Style Guidelines:

- Primary color: A deep, sophisticated navy (HSL: 220, 60%, 20% / Hex: #143052) that sets a luxurious tone and provides strong contrast for interactive elements in a dark theme.
- Background color: A very dark, subtly desaturated blue-black (HSL: 220, 15%, 8% / Hex: #121417), derived from the primary's hue, providing a plush, deep backdrop suitable for the dark mode first approach.
- Accent color: A vibrant, warm gold (HSL: 40, 90%, 55% / Hex: #E6B90E) used sparingly for calls-to-action, highlights, and subtle luxurious details to add prestige.
- Headings font: 'Playfair' (serif) for a modern, elegant, and high-end display aesthetic. Body font: 'Inter' (sans-serif) for clean readability and a contemporary feel across all content. Note: currently only Google Fonts are supported.
- Minimalist, line-based icons that maintain the clean and uncluttered aesthetic. Prioritize modern, abstract representations over literal ones to enhance the high-end feel.
- Spacious, grid-based layout with generous dark space around elements, emphasizing minimalism and luxury. The design will be fully responsive for optimal viewing across all device sizes, prioritizing a 'dark mode first' experience.
- Subtle, purposeful animations powered by Framer Motion, including smooth entrance effects for sections and interactive hover states (scale, shadow lift, image zoom) for property cards, enhancing user engagement without being distracting.
